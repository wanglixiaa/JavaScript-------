函数

位于return语句后的任何代码都永远不会执行

function gg(a,b){ return a+b; alert("dddd") } // gg(2,3);

alert("dddd")这行代码永远不会执行，而一个函数中可以包含多条return语句 return也可以不带有任何返回值，这种情况下，函数在停止后将返回undefined值。此用法一般在需要提前停止函数执行而又不需要返回值的情况下
